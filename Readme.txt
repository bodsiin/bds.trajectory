MODULE:

jmod.bds.trajectory

allows to control the jmod.presetmixpad module creating trajectories in the pad and receiving from the pad the size parameters.

SOFTWARE REQUIREMENTS:

To play it with ag.granular.suite as shown in the help file you will need Max/Msp 5, FTM 2.5.0 BETA 18 and Jamoma 0.5.2 beta 8.
Everything is alpha at the moment, so some things might occasionally blow up! Feel free to send suggestions and bug reports to bodsiin at gmail dot com

INSTALLATION NOTES:

Unzip and place all files somewhere in max/msp search path.
